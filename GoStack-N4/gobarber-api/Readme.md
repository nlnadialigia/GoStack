# :link: Como baixar o projeto

<br>

```js
// Clonar repositório
$ git clone https://github.com/nlnadialigia/GoStack.git

// Acessar diretório
$ cd GoStack/GoStack-N4/gobarber-api

// Instalar dependências
$ yarn

// Transpilar o projeto
$ yarn tsc


// Iniciar projeto
$ yarn dev:sever
```
